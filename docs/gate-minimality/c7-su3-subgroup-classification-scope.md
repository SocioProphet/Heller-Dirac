# C-7 — SU(3) Closed Connected Subgroup Classification Scope

Status: C-7 scope document; obligation not closed.  
Track: A2 gate-minimality, path beta convention.  
Claim level: scope/admissibility planning only.  
Depends on: `docs/gate-minimality/claim-boundary.md`, `docs/gate-minimality/a2-gate-minimality-structural-sketch.md`.

## Purpose

C-7 is the closed connected subgroup classification obligation for the adopted A2 convention:

```text
path beta + SU(3) + central Z/3 + cubic invariant
```

The obligation is not to re-derive the classification theorem. The obligation is to fix which closed connected subgroups of SU(3) must be considered, then determine which are admissible under the A2 gate-minimality criteria.

## Reference surface

Use Borel--de Siebenthal and standard compact Lie subgroup classification references as the classification background.

This document treats the classification theorem as external reference material. It does not reprove it.

## A2 admissibility filters

A subgroup candidate must be checked against at least these filters:

1. Closed and connected subgroup of SU(3).
2. Compatibility with the central Z/3 datum generated by `omega I_3`.
3. Compatibility with the active-sector cubic invariant selected by C-8.
4. Compatibility with the spatial faithfulness condition from the A2 structural sketch.
5. No collapse back to a path-alpha Spin(3) encoding unless explicitly marked as the C-9 alternative.

C-8 is not yet formulated. Any verdict depending on the cubic-invariant condition remains preliminary.

## Two SU(2) x U(1) embedding notes

For review discipline, the subgroup table separates two forms that can otherwise be conflated:

1. Standard block embedding: `S(U(2) x U(1))`, often identified with `U(2)` inside SU(3).
2. Principal SU(2)-type embedding extended by a compatible circle only if the extension is actually defined inside SU(3).

A later C-7 closure note must decide whether the second item is a genuine connected subgroup family under the adopted criteria or only a representation-theoretic comparison surface.

## Preliminary eliminations

The torus candidates are preliminary eliminations because they are abelian and cannot carry the full A2 gate object.

The standard `SU(2)` candidate is a preliminary elimination because it does not contain the central `Z/3` datum as scalar `omega I_3`.

The principal `SO(3)` candidate is a likely elimination because it does not carry the SU(3) center, but the exact statement should be made carefully in the C-7 closure proof.

`U(2)` / `S(U(2) x U(1))` is the major proper connected candidate that remains open pending C-8.

Full `SU(3)` is the adopted convention's expected admissible candidate.

## Work required to close C-7

A closure document must:

1. Cite the exact subgroup classification statement being used.
2. Normalize the candidate list so there is no duplicate counting of conjugate embeddings.
3. State the central Z/3 containment test precisely.
4. Apply the C-8 cubic-invariant condition once C-8 exists.
5. Mark every candidate as eliminated or admissible.
6. Preserve the claim boundary that this is still not an A2 theorem until all other proof obligations close.
