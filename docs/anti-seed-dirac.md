# Heller-Dirac Anti-Seed Register v1.0

Status: controlling negative register for the Heller-Dirac scaffold.
Claim level: boundary discipline only.

Heller-Dirac operates downstream of Heller-Godel framework anti-seed and PFK anti-seed. The entries below are Dirac-scaffold-specific extensions.

## A-HD-NC-001 — Reformulation is not proof

Recasting a Clay problem in spectral-triple, noncommutative-geometric, or Hopf-algebraic language is reformulation, not progress.

Correct statement: a reformulation may organize missing machinery, but theorem-grade progress requires native definitions, constructions, and proofs.

## A-HD-TM-001 — Modular flow is not automatically physical time

The thermal-time reading of modular flow requires precise hypotheses. Bisognano-Wichmann supplies one controlled correspondence in a specific vacuum wedge setting. Extensions beyond that scope require separate proof.

Correct statement: modular flow can be physically interpreted only when the hypotheses identifying it with physical evolution are established.

## A-HD-FT-001 — Axioms are not construction

A field-theory axiom system specifies a target structure. It does not by itself construct an example satisfying the target structure.

Correct statement: construction is complete only when the required objects and their properties are supplied in the native setting.

## A-HD-HA-001 — Hopf scaffold encoding is not a proof of analytic control

Hopf-algebraic organization can encode operations and counterterms. It does not by itself prove that a specific analytic or field-theoretic object exists or has the required control properties.

Correct statement: Hopf organization is apparatus; existence and analytic-control claims require domain-specific proof.

## A-HD-SP-001 — Analog spectral data is not target spectral data

If a spectral triple has spectrum resembling a target object's spectrum, the spectral data of the operator is not target spectral data unless an explicit identification is constructed.

Correct statement: analog spectra are fixture-grade or method-grade unless tied to the target by proof.

## A-HD-EX-001 — Fixture-grade computation is not Clay progress

## A-HD-FND-001 — Reference surface is not reproof

Heller-Dirac foundational identifiers `HD-FND-*` canonicalize reference surfaces for standard definitions and cited results. They do not reprove those results and do not convert them into new Heller-Dirac theorems.

Failure mode:


Correct statement:

```text

```

This discipline is permanent.
