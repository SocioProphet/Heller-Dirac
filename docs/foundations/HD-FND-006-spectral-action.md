## Statement surface

For a spectral triple `(A, H, D)`, the spectral action is:

```text
S_Lambda[D] = Tr f(D / Lambda)
```

where `Lambda > 0` is a cutoff scale and `f` is a suitable cutoff function.

The action depends on the spectrum of `D`. Under large-cutoff asymptotics it admits a heat-kernel expansion with coefficients determined by Seeley-DeWitt terms.

## Reading

The spectral action principle is the mechanism by which spectral data produces a classical action functional. In commutative cases it recovers geometric terms. In the noncommutative Standard Model construction it organizes gravitational and gauge-field action terms from spectral data.
